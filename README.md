# CodeAlpha Internship Tasks

This repository contains projects completed as part of the CodeAlpha Internship Program.

## Tasks Completed

### Task 2 - Stock Portfolio Tracker

A Python Tkinter application for managing stock holdings, calculating investment values, and exporting portfolio reports.

### Task 4 - Rule-Based Chatbot

A GUI-based chatbot developed using Python and Tkinter. The chatbot responds to user queries using predefined rules and conditional logic. It supports greetings, Python and AI-related questions, jokes, motivational quotes, date/time responses, chat history saving, and chat clearing functionality.

## Technologies Used

* Python 3
* Tkinter
* File Handling
* CSV
* Object-Oriented Programming
