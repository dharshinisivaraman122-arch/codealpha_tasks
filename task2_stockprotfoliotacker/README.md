# Stock Portfolio Tracker

## Description

Stock Portfolio Tracker is a Python-based GUI application developed using Tkinter. It allows users to manage stock holdings, calculate investment values using predefined stock prices, and maintain a simple investment portfolio.

This project was developed as part of the CodeAlpha Internship Program to demonstrate Python fundamentals, GUI development, file handling, and object-oriented programming concepts.

## Features

* Add stocks with quantity
* Hardcoded stock price dictionary
* Automatic investment value calculation
* Portfolio table display
* Running total portfolio value
* Remove selected holdings
* Clear entire portfolio
* Export portfolio report as TXT file
* Export portfolio report as CSV file
* Input validation and error handling
* User-friendly Tkinter interface

## Technologies Used

* Python 3
* Tkinter
* CSV Module
* File Handling
* Object-Oriented Programming (OOP)

## Concepts Demonstrated

* Dictionaries
* Functions
* Classes and Objects
* Arithmetic Operations
* Input and Output
* File Handling
* GUI Development with Tkinter

## How to Run

1. Download or clone the repository.
2. Open a terminal in the project folder.
3. Run the following command:

```bash
python stock_tracker.py
```
