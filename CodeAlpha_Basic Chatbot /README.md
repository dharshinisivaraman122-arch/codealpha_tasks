# Rule-Based Chatbot

## Description

Rule-Based Chatbot is a Python-based GUI application developed using Tkinter. The chatbot responds to user messages using predefined rules and conditional statements. It can answer common questions, provide information about Python, Artificial Intelligence, and Machine Learning, tell jokes, share motivational quotes, and display the current date and time.

This project was developed as part of the CodeAlpha Internship Program to demonstrate Python fundamentals, conditional logic, functions, file handling, and GUI development.

## Features

* Interactive chatbot interface
* Greeting and conversation support
* Information about Python, AI, and Machine Learning
* Random jokes and motivational quotes
* Current date and time responses
* Save chat history as a TXT file
* Clear chat functionality
* Input validation
* User-friendly Tkinter GUI

## Technologies Used

* Python 3
* Tkinter
* File Handling
* Datetime Module
* Random Module

## Concepts Demonstrated

* If-Else Conditions
* Functions
* Event-Driven Programming
* Input and Output
* File Handling
* Lists and Strings
* GUI Development with Tkinter

## How to Run

1. Download or clone the repository.
2. Open a terminal in the project folder.
3. Run the following command:

```bash
python chatbot.py
```
